void selector_screen::selector ( )

Function to create and initialize the selector screen.

This function creates a tab view object and adds two tabs to it: "Autonomous select" and "Drive select". It also adds event callbacks to the tabs for updating the UI. The function creates labels, switches, and a dropdown list for selecting alliance color, skills, and autonomous mode. It also creates a label and a roller for selecting the current drive mode.
