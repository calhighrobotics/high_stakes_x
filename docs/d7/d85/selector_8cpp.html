static void drive_update ( lv_event_t * e)
static

Updates the drive mode based on the selected option in the drive roller.

This function is called when the value of the drive roller is changed. It retrieves the selected option from the drive roller and updates the drive mode accordingly. The updated drive mode is then displayed on the drive label.

Parameters
eThe event object associated with the drive roller.
