enum v5_device_e

Gets the type of device on given port.

Returns
The device type as an enum.

Example

#define DEVICE_PORT 1
void opcontrol() {
while (true) {
v5_device_e_t pt = get_plugged_type(DEVICE_PORT);
printf("device plugged type: {plugged type: %d}\n", pt);
delay(20);
}
